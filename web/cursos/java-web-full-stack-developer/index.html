    El programa por semanas

    Tu primera web

    Semanas 1 a 4

    Metete de lleno en el mundo de la programación adquiriendo conocimientos sobre el Front-End. Aprenderás HTML5, CSS3 y JavaScript. Poco a poco implementarás las mejores prácticas recomendadas.

    Mobile web

    Semanas 5 a 8

    Aprendé a hacer webs que se vean bien en todos los dispositivos. En este módulo diseñarás y crearás una Mobile Web y aprenderás a usar Bootstrap. Empiezan las sesiones de Coaching para trabajar tus Habilidades Personales.

    Backend con JAVA

    Semanas 9 a 16

    Creá el juego “Batalla Naval” para web, y aprendé a manejar los perfiles de los jugadores y la puntuación. Usarás Java y Spring Boot Framework. También te ayudaremos con el CV, las entrevistas, LinkedIn, networking...

    ¿A quién va dirigido este programa?

    Egresados secundarios

    ¿Terminaste el secundario y no estás dispuesto a embarcarte en años de estudios universitarios? Querés tener una salida laboral rápida y bien remunerada? Este es un programa intensivo de Java, HTML5, CSS3 y Javascript. Además de aprender metodologías ágiles de desarrollo y programación estructurada, todo lo que harás será sobre proyectos reales. Así que tu trabajo como desarrollador comienza con la primer clase.

    Universitarios recién graduados o próximos a graduarse

    ¿Acabás de terminar la universidad o estás cursando? Con este programa vas a adquirir las habilidades necesarias para ganar experiencia laboral, horas de desarrollo, y encontrar un empleo de forma inmediata en una empresa o bien lanzar tu propio negocio.

    Los profesionales que quieran hacer un reskilling

    ¿Querés hacer un giro en tu carrera actual? Sabes COBOL, ABAP, Natural, Visual y otros tantos lenguajes, pero sentís que no estás subido a la ola digital? Aprender a desarrollar webs y aplicaciones para móviles es una gran idea. Se trata de una profesión altamente calificada y, al terminar el programa, serás un firme candidato a darle un envión a tu proyección laboral y encontrar un empleo más atractivo y mejor remunerado. Tenemos beneficios especiales para los profesionales mayores de 40 años.

    Emprendedores

    Si estás pensando en emprender, este programa te ayudará a adquirir las habilidades necesarias para construir un MVP (producto mínimo viable) de tu negocio digital, desarrollando tanto el front como el back end.
